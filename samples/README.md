# Sample PDFs

This directory contains a public demo corpus for grounded Q&A:

- Source: official Federal Reserve FOMC statement PDFs
- Corpus path: `samples/pdfs/`
- Coverage: 2024-01-31 through 2025-12-10

## Example Questions

Use these to demonstrate timeline summarization and grounded retrieval:

1. Summarize how the Fed's language about inflation changed from January 2024 through December 2025.
2. List the meetings where the target federal funds rate changed, with the old and new range if stated in the documents.
3. Summarize how the Committee described labor market conditions over time.
4. Identify the meetings where the statement language was materially unchanged versus meaningfully revised.
5. Summarize the main policy concerns that appear repeatedly across the full statement history.
6. Build a concise timeline of how the Fed described inflation, employment, and growth across these statements.
7. Which statements mention stronger confidence or reduced confidence about inflation moving toward target?
8. Summarize the progression from restrictive policy language toward any easing language, if present.

## Notes

- These PDFs are public and safe to share in demos.
- This corpus works well for questions that require comparing many documents over time rather than extracting a single fact.
